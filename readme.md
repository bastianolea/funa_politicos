# Funa Políticos

_Proyecto en etapa temprana de desarrollo_

Proyecto inspirado por el [visualizador de casos de corrupción en Chile](https://bastianolea.rbind.io/blog/app_corrupcion_chile/). 

Este proyecto se propone reunir en un sólo lugar casos periodísticos de personajes públicos (políticos, empresarios) que revelen abuso de poder, corrupción, mentiras, asociaciones sospechosas y delitos.

El objetivo es dar visibilidad a investigación periodística y judicial sobre personajes que ostentan poder y cuyas malas prácticas suelen pasar desapercibidas.

### Descargo de responsabilidad
No se emiten opiniones personales en este proyecto. Todos los casos son incluidos bajo el criterio de su aparición en prensa. En este sentido, no se trata de difamación, injuria ni calumnia, sino de un repositorio de casos que han sido investigados y publicados por medios de comunicación. Cualquier contribución de casos será bien recibida.

## Datos
Los datos se encuentran en la carpeta `datos/` en formato Excel.

## Hoja de ruta
- Recolectar una cantidad base de casos
- Levantar base de datos con casos
- Desarrollar plataforma de visualización de casos alimentada por la base de datos
- Desarrollar sistema de contribución pública de casos por medio de la plataforma
